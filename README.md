# Event Management Backend

This is the backend API for the Event Management system built with Node.js, Express, and MongoDB.

## Features
- CRUD operations for events
- User registration and login (with JWT or session)
- Event registration and cancellation
- Admin-only routes for event creation/deletion
- MongoDB data storage

## Prerequisites
- Node.js (>=18)
- npm or yarn
- MongoDB Atlas / Local MongoDB instance
- Optional: Cloudinary account (for event images)
